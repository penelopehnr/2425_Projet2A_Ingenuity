### 1. Connect Filezilla to the RasberryPi0

### 2. Launch VsCode and cmd

### 4. run as admin code server.py on cmd
    - sudo python3 server.py

### 5. run client code on VsCode
    - CRTL+F5 on client.py

### 6. Follow program requests
